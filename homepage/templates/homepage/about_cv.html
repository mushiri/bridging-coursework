About CV


The main purpose of the coursework was to create a CV. I had some difficulty thinking about the design at first.

I followed the same approach and mechanisms of the blog application.

The only difference in the CV application is that I added an RTF (Rich-text field) editor called CKeditor.

This editor is compatible with Django, and the reason I added it because it contains tools where I can align and change the text.

In addition, instead of adding 'posts', I call each block of information a 'card'. I can use the 'CRUD' mechanism on cards.
