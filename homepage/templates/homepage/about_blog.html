About Blog


The coursework was announced on May 15th. The purpose of the coursework was to create a CV using Django.

Everything was new, and I did not have much knowledge regarding web application nor Django. The first tutorial that I started was 'Django Girls Tutorial'.

I took this opportunity to learn more about web development.

The tutorial was aimed for beginners, and the resulting application was a blog.

I learnt the basics of web development as well as how Django works.

After experiencing with Django and learning more about it, I followed a different approach.

The tutorial followed a "function-based views" rather than "class-based views" which I have used.

Using "class-based views" is much easier in my opinion. Since all my projects use 'CRUD' mechanism.

In other words, Creating, Reading, Updating, and Deleting. Using Django's built-in templates made it easier when using "class-based views".
